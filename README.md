🔬 Featured work
`neural-nets-vs-regular-languages` — Bachelor's Thesis
> An empirical study of \*\*why some regular languages are harder for neural networks to learn than others.\*\* I investigate the link between the \*\*Boolean sensitivity\*\* of a language and the \*\*structural complexity of its minimal automaton (DFA)\*\* — backed by from-scratch browser demos: a backprop + Adam trainer and a parity-DFA hypercube visualizer. Where learning theory and geometry meet.
`ia-marketing` — AI tooling for paid-media analysis
> A Python system on the \*\*Google Ads API\*\* (OAuth2) that pulls campaign data and applies domain rules to surface real decisions — impression-share saturation, account-level CPA/ROAS, brand-campaign targeting. Real client, real data.
`tamandua-compiler` — A full compiler in Java
> A complete compiler with \*\*Java + CUP + JFlex\*\*: lexing, parsing, an attributed grammar for semantic analysis, and code generation to a stack machine. The project that teaches you what your languages actually do underneath.
